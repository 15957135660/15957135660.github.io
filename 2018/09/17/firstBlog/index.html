title: how to start a personal blog

date: 2018-09-17 16:50:08
tags：

​ 前言：一直想自己搭建一个自己的博客，观感里，觉得这是比较麻烦的事，包括搭建服务器，申请域名等等工作。偶然发现搭建自己的博客并不困难，于是，基于网上的教程，开始了自己博客的搭建。

​ 参考文章：#分享# 自己搭建免费博客 Hexo + Github 以及 Coding.net

​

​ 搭建注意事项：

1.参考文章每一步，搭建基本没有大问题，在第6步 Hexo 与 GitHub page 联系起来 时，文章内容有笔误

​ 输入git config –global user.name “Githhub绑定的邮箱” 应为 git config –global user.email “Githhub绑定的邮箱”

2.第7步修改 _config.yml 时，千万注意 ： 后面的空格，没有空格，这里会影响博客的部署

3.增加一份资料 Hexo常见问题解决方案
